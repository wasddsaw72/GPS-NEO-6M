# GPS-NEO-6M
GPS module with LCD1602 and STM32 Nucleo f446

Для проекта ипользовались следующие комопненты: 
1) STM32 Nucleof446
2) lcd 1602 на плате под arduino разьём (LCD Keypad Shield)
3) GPS Module NEO-6M
